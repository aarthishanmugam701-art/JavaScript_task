# JavaScript_task


## Overview

This repository contains three basic JavaScript programs:

1. Button Color Change
2. Greeting Alert Message
3. Simple Addition Calculator

These programs demonstrate fundamental JavaScript concepts such as event handling, conditional statements, and user input processing.

---

## 1. Button Color Change

### Description

A button changes its background color when clicked.

### Features

* Uses JavaScript event handling.
* Dynamically updates button style.

### Technologies Used

* HTML
* JavaScript

---

## 2. Greeting Alert Message

### Description

Displays a greeting message based on the current system time.

### Features

* Detects current hour.
* Shows Good Morning, Good Afternoon, or Good Evening.

### Technologies Used

* HTML
* JavaScript

---

## 3. Simple Addition Calculator

### Description

Adds two numbers entered by the user and displays the result.

### Features

* Accepts user input.
* Performs addition.
* Displays the calculated result.

### Technologies Used

* HTML
* JavaScript

---

## Learning Outcomes

* Understanding JavaScript functions.
* Using DOM manipulation.
* Handling button click events.
* Working with conditional statements.
* Performing basic arithmetic operations.
